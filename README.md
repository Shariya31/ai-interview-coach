# AI Interviewer

Small, iterative project to build an AI-powered interviewer web app.

## Structure
- `/server` - Node + Express backend and AI abstraction
- `/client` - React + Vite frontend
- `/docs` - design, APIs, notes

## Quickstart
1. Start server: `cd server && npm install && npm run dev`
2. Start client: `cd client && npm install && npm run dev`
